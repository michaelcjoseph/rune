# Front-End Review

## Task

Review a cockpit sidebar change that adds a per-project status control.

## Good designer output

Findings:

- Blocking: none.
- Medium: the disabled state uses color alone; add an icon or text state so status is not
  conveyed only by hue.
- Low: the action label wraps awkwardly below 360px; tighten the copy or use the existing
  icon button pattern.

Notes:

- The control matches the cockpit's compact operational style.
- The layout preserves scanability across products and projects.
- No marketing-style hero, decorative card nesting, or oversized explanatory copy was added.
